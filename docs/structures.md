## Project Structure

```
- project/
  - src/
    - services.py
    - utilities.py
  - tests
    - test_services.py
    - test_utilities.py
  - main.py
```
**src/** is the source code packages of the project, the main algorithm is stored in **services.py** file while the helper utilities
is in the **utilities.py**

**test/** is the directory packages for the unittest files, **test_services.py** is a unittest file to test the algorithm and the **test_utilities.py**
is the file to test the helper utilities and it's function
